This site exemplifies many of my fundamental philosophies. Allow me to elaborate.
First and foremost, I am a strong proponent of simple, flat design. No gradients, no shadows, and minimal color schemes. True to form, I designed this site with only four colors, large margins for easy focus, and minimal moving parts.

Responsive design is always a priority. With the advent of CSS media queries, there are no longer any valid excuses for not designing with phones and tablets in mind. On a phone? Good for you. For my desktop users, reduce the width of your browser window and watch the magic happen.

Lastly, this site is built with lots of love on Django, a Python framework. I am determined to never build another static website again. Check out the Github Repo to see the code.
